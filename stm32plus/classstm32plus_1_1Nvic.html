Detailed Description

F0 implementation of the NVIC.

F4 implementation of the NVIC.

F1 implementation of the NVIC.

The F0 NVIC is simpler than the F1/F4. Channel priorities are supported in a range of 0..3 where zero is the highest but there are no priority groups or sub-priorities. Method interface compatibility is maintained between this and the F1/F4 implementation.

The NVIC methods allow us to configure the number of bits available for priority and subpriority. The default is all 4 bits for priority and none for subpriority. Other methods allow us to enable/disable the IRQ for a given peripheral.
