Detailed Description

template<uint8_t TChannelNumber>
class stm32plus::DmaInterruptFeatureEnabler< TChannelNumber >

Helper class to enable only the desired interrupts in the NVIC. This will be fully specialised for each DMA peripheral and channel

Template Parameters:
TChannelNumberThe number of the DMA channel (1..5)

Helper class to enable only the desired interrupts in the NVIC. This will be fully specialised for each DMA peripheral and channel

Template Parameters:
TDmaNumberThe number of the DMA peripheral (1..2)
TChannelNumberThe number of the DMA channel (1..7) or (1..5)

Helper class to enable only the desired interrupts in the NVIC. This will be fully specialised for each DMA peripheral and channel

Template Parameters:
TDmaNumberThe number of the DMA peripheral (1)
TChannelNumberThe number of the DMA channel (1..7)
