Detailed Description

template<uint8_t TExtiNumber>
class stm32plus::ExtiInterruptEnabler< TExtiNumber >

Helper class to enable only the desired interrupts in the NVIC. This will be fully specialised for each EXTI peripheral

Template Parameters:
TExtiNumberThe number of the Exti peripheral (0..19 | 22)
