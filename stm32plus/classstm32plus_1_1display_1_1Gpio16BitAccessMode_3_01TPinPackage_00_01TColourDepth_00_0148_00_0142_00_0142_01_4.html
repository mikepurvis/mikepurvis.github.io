Detailed Description

template<class TPinPackage, ColourDepth TColourDepth>
class stm32plus::display::Gpio16BitAccessMode< TPinPackage, TColourDepth, 48, 42, 42 >

Specialisation for 48Mhz HCLK, 64K colour mode, 42ns low and 42ns high write cycle periods. The 20.8ns HCLK means that we actually achieve 41.6ns (+/- clock accuracy).

This driver is intended for use with the 48MHz products, for example the F0 Discovery board.

This is for 18, 24 bit colour depths. The 64K specialisation should be preferred for 16-bit colour depths as it contains more optimisations for increased speed. The compiler should automatically pick the correct specialisation for you.
