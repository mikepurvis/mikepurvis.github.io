Detailed Description

template<uint32_t Line>
class stm32plus::ExtiPeripheral< Line >

Template implementation for the EXTI lines.

Template Parameters:
IndexThe index (0..15) of the EXTI line.
LineThe EXTI line e.g. EXTI_Line0
IrqThe corresponding IRQ for the EXTI line.

Exti 0..15 configuration differs only slightly so we define as a template with integral parameters and then declare typedefs for the user usage.

The ExtiInterruptEventSender member can be used to subscribe to the interrupts. See the 'exti' example for details of how to do this.
