Detailed Description

template<class TGraphicsLibrary>
class stm32plus::display::JpegDecoder< TGraphicsLibrary >

JPEG decoder. Implements the callback from the picoJpeg decoder to write 8x8 blocks to the screen.

Either call decode() to decode the whole JPEG or call beginDecode() then endDecode() if you need access to the image dimensions
