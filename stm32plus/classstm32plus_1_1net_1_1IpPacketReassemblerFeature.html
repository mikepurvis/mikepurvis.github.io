Detailed Description

Handler for fragmented IPv4 packets. Fragmented packets are held in a linked list until either the full packet arrives or one of the parameterised limits is hit. Management of the incoming packets is done according to the algorithm described in RFC815 with the 'hole' descriptors existing inside the holes themselves.
