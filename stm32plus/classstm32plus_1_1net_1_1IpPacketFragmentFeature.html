Detailed Description

Class to take a netbuffer and generate collection of other netbuffers that contain the input data fragmented into parts small enough to fit into the link layer's max MTU size.

The original netbuffer's data is referenced by pointers in the fragment buffers so very little additional memory is required. The original netbuffer is added as a reference to the last fragmentso that any higher layer that is tracking outgoing packets by waiting for a netbuffer they created to be sent will function correctly.
