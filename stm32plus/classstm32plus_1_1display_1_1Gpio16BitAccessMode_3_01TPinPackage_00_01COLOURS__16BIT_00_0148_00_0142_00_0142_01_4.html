Detailed Description

template<class TPinPackage>
class stm32plus::display::Gpio16BitAccessMode< TPinPackage, COLOURS_16BIT, 48, 42, 42 >

Specialisation for 48Mhz HCLK, 64K colour mode, 42ns low and 42ns high write cycle periods. The 20.8ns HCLK means that we actually achieve 41.6ns (+/- clock accuracy). This driver is intended for use with the 48MHz products, for example the F0 Discovery board.
