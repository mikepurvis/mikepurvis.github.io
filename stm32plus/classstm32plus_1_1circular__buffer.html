Detailed Description

template<typename T>
class stm32plus::circular_buffer< T >

Template class to manage a circular buffer of types. Separate read and write pointers are maintained. Methods are provided to read from and write to the buffer. More methods are provided to find out how much read space and write space is available without overrunning the other position.
