Detailed Description

template<uint8_t TPrescaler>
struct stm32plus::AdcClockPrescalerFeature< TPrescaler >

Feature class to allow changing the prescaler for the ADC analog clock. The ADC clock on the F1 runs off PCLK2 divided by a prescaler. You specify the prescaler: 2/4/8 as a template parameter and the code selects the correct constant at compile time.

Feature class to allow changing the prescaler for the ADC analog clock. The ADC clock on the F4 runs off APB2 divided by a prescaler. You specify the prescaler: 2/4/8 as a template parameter and the code selects the correct constant at compile time.
