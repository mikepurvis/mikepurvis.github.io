Detailed Description

template<class TDatalinkLayer>
class stm32plus::net::Arp< TDatalinkLayer >

Network layer feature that implements the Address Resolution Protocol (ARP) ARP is used as to convert one address form into another. In our implementation we will support converting MAC addresses into IP addresses.
