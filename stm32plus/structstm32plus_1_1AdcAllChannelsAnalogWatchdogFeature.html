Detailed Description

template<uint16_t TLow, uint16_t THigh>
struct stm32plus::AdcAllChannelsAnalogWatchdogFeature< TLow, THigh >

Support for configuring the analog watchdog feature. The AWD allows you to trigger an interrupt if a voltage crosses pre-defined upper or lower bounds. This feature allows you to configure all enabled channels for monitoring.

This feature would normally be combined with AdcInterruptFeature in your ADC declaration.
