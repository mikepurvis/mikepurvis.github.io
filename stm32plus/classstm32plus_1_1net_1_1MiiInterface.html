Detailed Description

template<class TPhysicalLayer, class TPinPackage>
class stm32plus::net::MiiInterface< TPhysicalLayer, TPinPackage >

Class for use of the MAC in MII mode on the F107. The only available pin package is MacDefaultPinPackage.

Class for use of the MAC in MII mode on the F4.

Template Parameters:
TPinPackageThe type containing the enumeration of pins that we are to use for RMII. e.g. MacDefaultPinPackage
