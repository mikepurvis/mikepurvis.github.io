Detailed Description

template<class TSpi>
class stm32plus::spiflash::Base< TSpi >

Base template for SPI commands. TSpi is a template parameter instead of an instance of "Spi" so that a wrapper can be used to emulate SPI over GPIO or defer to a DMA channel that's connected to the SPI peripheral.
