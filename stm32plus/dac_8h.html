Detailed Description

This file gets you access to the STM32 DAC peripheral. Functionality is provided to access both DAC channels either synchronously or via DMA.

A driver is also provided for the CS43L22 peripheral that can be found onboard the STM32F4DISCOVERY board sold by ST Microelectronics.
