Detailed Description

template<Orientation TOrientation, ColourDepth TColourDepth, class TAccessMode, class TPanelTraits>
class stm32plus::display::R61523< TOrientation, TColourDepth, TAccessMode, TPanelTraits >

Generic R61523 template. The user can specialise based on the desired colour depth, orientation and access mode. The TPanelTraits template parameter allows customisation of the driver to cope with panels with slightly different characteristics.
