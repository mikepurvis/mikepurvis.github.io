Detailed Description

template<class TFsmc>
class stm32plus::display::Fsmc16BitAccessMode< TFsmc >

FSMC implementation for TFT LCD panels controlled by an 8080 interface.

The 8080 LCD is assumed to be in 16 bit mode connected to the 16 data pins, RW, RD, NE.

The RS (register select) operation is achieved by using an additional address line. e.g. A16. A write to an address that keeps A16 low will active RS (usually active low). A write that sets A16 high will set RS high and achieve a data write to the controller.
