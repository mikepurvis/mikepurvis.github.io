template<uint8_t TUsartNumber>
class stm32plus::UsartInterruptFeatureEnabler< TUsartNumber >

Helper class to enable only the desired interrupts in the NVIC. This will be fully specialised for each USART peripheral

Template Parameters:
TUsartNumberThe number of the USART peripheral (1..2)
