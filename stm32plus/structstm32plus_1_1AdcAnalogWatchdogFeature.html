Detailed Description

template<uint16_t TLow, uint16_t THigh>
struct stm32plus::AdcAnalogWatchdogFeature< TLow, THigh >

Support for configuring the analog watchdog feature. The AWD allows you to trigger an interrupt if a voltage crosses pre-defined upper or lower bounds.

This is a base class that supports the threshold setting ability common to all MCUs
