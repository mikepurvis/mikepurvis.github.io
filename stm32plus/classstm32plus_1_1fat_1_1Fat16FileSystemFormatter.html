Initialise the class and do the format.

Check the error provider status to find out whether the format worked. The format operation prepares the FAT data structures. It does not do a low-level format of the device.

Parameters:
[in]blockDeviceThe device to format.
[in]firstSectorIndexThe zero-based index of the first sector of the partition, or the device as a whole if partitions are not supported.
[in]numSectorsThe total number of sectors to format on this device.
[in]volumeLabelThe new name for the volume, max 11 characters.
