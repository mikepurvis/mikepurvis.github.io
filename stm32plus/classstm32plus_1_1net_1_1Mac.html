Detailed Description

template<class TPhysicalLayer>
class stm32plus::net::Mac< TPhysicalLayer >

Template class for the MAC. Although the PHY is generally considered to be in the same network layer (#2, datalink) as the MAC they are not independent devices. The MAC is wholly dependent on, and strongly linked to its PHY. Therefore we implement the PHY as a base of the MAC.

Template Parameters:
TPhyThe type of PHY, e.g. KSZ8051MLL
TInterfaceModeThe type of interface MiiInterface or RmiiInterface
