Detailed Description

template<class TPhysicalLayer, template< class > class... Features>
class stm32plus::net::DatalinkLayer< TPhysicalLayer, Features >

Template feature-holder for features that live in the datalink network layer, e.g. MAC. The layer below this is the physical layer and the layer above is the network layer.
