Detailed Description

Base class for anything that needs to use the shared port ranges. This class manages handing out Ephemeral ports. Ephemeral ports are in the 61440-65535 (4K) range. This is significantly less than the 16K ports you get with a typical PC stack but in this MCU we do not have the resources to handle 16K simultaneous ports.

This class requires 512 bytes of SRAM for the port bitmap.

You can customise the above numbers using the ip_ephemeralPortCount variable in the parameters class.
