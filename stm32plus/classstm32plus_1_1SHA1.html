Detailed Description

SHA1 implementation for the F4 processor. Basic polling operation is supported here. After construction, use it like this.

1. Call update() zero or more times to add 32-bits of data to the hash computation. 2. Call finish() for the last 32 bits or less of data that you have and receive the digest. 3. To use again, call reset() and go back to step (1).

Adaptation of the SHA1 algorithm credited in the comment above.
